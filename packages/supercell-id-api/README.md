# @brawlbase/supercell-id-api

Allows you to interact with Supercell's Supercell ID API, and allow you to authorize users
with Supercell ID!

## Installation

```bash
npm install @brawlbase/supercell-id-api
# or
pnpm install @brawlbase/supercell-id-api
# or
yarn add @brawlbase/supercell-id-api
# or
bun install @brawlbase/supercell-id-api
```
